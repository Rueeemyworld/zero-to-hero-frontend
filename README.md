## 为什么学前端
- 兴趣。从小到大我对自己不会的东西都感兴趣，有时候觉得我就是背上绑着鱼竿的那头驴，而未知就是那根拴在鱼线上勾引我的胡萝卜，终其一生我都在追寻这种若即若离的美。今年，我考了美国调酒师证、证婚证、自媒体运营、拍人像、学剪辑，直播、学习系统性健身。此前，我自学了做美甲、卖蛋糕卖饰品卖衣服。在这种不断折腾的道路上我获得了很多全新的体验和对社会的理解，也同样在意识到世界太大后发现：人这一生终究要做取舍，在不断地选择题中发掘出自己的道路，而不是随波逐流又或者既要又要。
- 于是这个文档便产生了。这也是在为即将开始的2026年制定一个我的全新人生算法的章程，同样也是今年经历大起大落后认识的血泪经验————
  1.不要一直埋头向前跑，真正能更快到达终点的是不断停下来复盘
  2.团队协作的力量永远大于个人，在这里欢迎各位朋友对我的学习以及教程提供建议，感激不尽
  3.成功最简单又最难的路径就是重复，重复，再重复
  4.学前端和健身一样，没人嘲笑那个白斩鸡身材的你，我们看不起的是第二天不再来的人
